screen · check-in post-run
Pantalla del check-in de 20 segundos. Útil para reels, demos, posts educativos.

Instrucciones para uso

  1. Screenshot directo: abrí este HTML en Chrome, hacé zoom al 100%, screenshot del phone que quieras (Cmd+Shift+4 en Mac, seleccioná el phone).
  2. Mejor calidad: abrir en Safari → File → Export as PDF → recortar.
  3. Para Figma: el diseñador puede tomar este HTML como referencia exacta y reconstruir en Figma con los componentes del brandbook.
  4. Para Instagram: screenshot del phone (sin fondo) → pegar sobre fondo verde musgo en el slide del carousel.
