# FlowRun

Coach AI de trail running. Corré al ritmo correcto. Disfrutá más.

Proyecto independiente desarrollado en Argentina.

## Estructura

- `landing/` — Landing page pública (`index.html`).
- `app/` — Código de la app (en construcción).
- `brand/` — Brandbook y assets de marca (logos en SVG).
- `mockups/` — Prototipos HTML de pantallas, planes, onboarding y entregables.
- `instagram/` — Posts y contenido social.
- `discovery/` — Documentación de producto: visión, pricing, terms, schemas, planes de entrenamiento, estrategia de contenido.
- `archive/` — Versiones previas (Ritmo Real) y prototipos antiguos.

## Landing

Abrir `landing/index.html` en el navegador.
Self-contained: usa Google Fonts via CDN, sin build step.
